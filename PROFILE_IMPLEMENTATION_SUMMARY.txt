   4. Verify success message

   Should return 2 rows:
   • avatars (public: true, file_size_limit: 5242880)
   • resumes (public: false, file_size_limit: 5242880)

┌──────────────────────────────────────────────────────────────────────────────┐
│ 📊 API ENDPOINTS                                                             │
└──────────────────────────────────────────────────────────────────────────────┘

   GET /api/v1/profile
   • Fetch current user's profile
   • Returns: user, profile, candidateProfile
   • Auth: Required

   PATCH /api/v1/profile
   • Update profile fields
   • Supports partial updates
   • Auto-creates records (upsert)
   • Auth: Required

   POST /api/v1/upload
   • Upload avatar or resume
   • FormData: { file: File, type: 'avatar' | 'resume' }
   • Returns: { url, fileName, size, type }
   • Auth: Required
